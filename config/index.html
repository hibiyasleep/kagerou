이유

OverlayPlugin이 같은 오버레이가 두 개 이상의 창을 띄우는 것을 허용하지 않아서, 직접 오버레이를 다시 실행해줘야 합니다.

설정을 변경하지 않았어도, 위의 이유로 오버레이를 다시 사용하려면 아래의 단계를 따라야 합니다.

방법

  1. ACT 메인 창으로 갑니다.
  2. Plugins > OverlayPlugin.dll > kagerou
  3. 오른쪽 아래의 Reload overlay 버튼을 누릅니다.
  4. 설정 창이 자동으로 닫히고, 오버레이가 다시 나타납니다.
